# Student Marks Management System

A simple Python project to manage student marks using Object-Oriented Programming concepts.

## Features
- Create a student with a name and marks
- View current marks
- Update marks with validation
- Encapsulation using private variables
- User-friendly command-line menu

## Concepts Used
- Classes and Objects
- Encapsulation (private attributes)
- Getters and Setters
- `__str__` method
- While loop & conditionals
